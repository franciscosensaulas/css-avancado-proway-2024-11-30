Media types

O media type em CSS serve para definir estilos específicos para diferentes tipos de dispositivos ou condições de exibição. Ele permite que você adapte o design e o layout de uma página de forma responsiva, dependendo do contexto de visualização, como tamanho da tela, tipo de dispositivo ou meio de impressão.

A regra @media print define estilos específicos para quando a página for impressa.

A regra @media screen and (min-width: 1024px) aplica estilos para telas com largura de 1024px ou mais, como desktops.

A regra @media screen and (max-width: 600px) aplica estilos para telas com largura máxima de 600px, aplicada normalmente para telas pequenas, como smartphones.
